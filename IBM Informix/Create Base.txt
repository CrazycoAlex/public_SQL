БД: reportsdb

Создание пространства (чанка) repdbspc для БД:
В C:\Program Files (x86)\Informix\storage
Создать файл: repdb_chunk1
Создать ссылку на файл: mklink repdb_s01c01 repdb_chunk1
В dbacces создать пространство с использованием ссылки на чанк 
(таким образом возможно в будущем безболезненно перенести чанк на другое место):
onspaces -c -d repdbspc -p repdb_s01c01 -o 0 -s 52428800

Проверка доступных пространств: onstat -d
Пометка пространства как архивированного: onbar -b -F

Создание пространства (чанка) repdbtmp для БД:
В C:\Program Files (x86)\Informix\storage
Создать файл: repdb_temp1
Создать ссылку на файл: mklink repdb_t01 repdb_temp1
В dbacces создать пространство с использованием ссылки на чанк: 
onspaces -c -d repdbtmp -t -p repdb_t01 -o 0 -s 10485760

Проверка доступных пространств: onstat -d
Пометка пространства как архивированного: onbar -b -F
